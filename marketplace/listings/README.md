# Marketplace Listings

This directory contains individual listing metadata files for products in the Ninobyte marketplace.

## Structure

Each product has a listing file:
```
listings/
├── senior-developer-brain.json
├── [future-product].json
└── ...
```

## Listing Format

See `../marketplace.json` for the catalog schema and `../qa/QA_CHECKLIST.md` for acceptance criteria.
