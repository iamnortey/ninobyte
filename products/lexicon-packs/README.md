# Lexicon Packs

This directory contains domain-specific vocabulary and terminology packs.

## Status

No lexicon packs released yet. Planned for future versions.

## Structure

Each lexicon pack will be a subdirectory:
```
lexicon-packs/
├── [domain-name]/
│   ├── README.md
│   ├── lexicon.json
│   ├── examples/
│   └── ...
└── ...
```

## Development Guidelines

- Follow [THREAT_MODEL.md](../../docs/architecture/THREAT_MODEL.md) security requirements
- Include usage examples
- Document terminology sources
